# Movie Diary Web App

A **dynamic movie diary web app** built with **JavaScript**, **HTML5**, and **CSS**, utilizing the **TMDB API** to fetch real-time movie data.

## Features
- **Search functionality** to find movies by title.
- Displays **movie details** including title, overview etc.
- **Responsive design** for optimal viewing on all devices.

## Tech Stack
- **JavaScript**, **HTML5**, **CSS**
- **API Integration:** TMDB API
- **Tools:** VS Code, Git, GitHub

## Sample Usage
1. Open `index.html` in a browser.
2. Enter a movie title in the search bar.
3. View the movie details displayed below the search bar.
